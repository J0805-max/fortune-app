# 今日运势 🍀

每日星座运势 + 黄历推送 · 天秤座专属

## 如何使用

### 📱 安卓手机
1. 在浏览器中打开本应用地址
2. 点浏览器菜单 → **添加到主屏幕**
3. 桌面上出现「运势」图标，像普通 App 一样用

### 💻 Windows 电脑
1. 用 Chrome/Edge 打开本应用地址
2. 地址栏右侧会出现安装图标，点击安装即可

### ⚙️ 首次设置
1. 打开 App → 点底部 **设置**
2. 设置你想要的 **推送时间**（默认 7:30）
3. 如需实时数据：在 **天行数据 (tianapi.com)** 免费注册 → 获取 API Key

## 📡 数据来源

| 数据类型 | 来源 | 说明 |
|---------|------|------|
| 星座运势 | 天行数据 (tianapi.com) | 免费注册即用 |
| 黄历信息 | 天行数据 | 同上 |
| 农历日期 | 香港天文台 (hko.gov.hk) | 免费公开数据 |

所有数据均为真实可查的公开来源。App 内「设置 → 数据来源」可查看详情。

## 🛠️ 技术栈

- PWA (Progressive Web App)
- HTML5 + CSS3 + JavaScript (原生)
- Service Worker (离线缓存 + 推送通知)
- LocalStorage (本地数据缓存)
