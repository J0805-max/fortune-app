# 今日运势 App — 设计文档

> 版本: v1.0
> 日期: 2026-06-21
> 星座: 天秤座 (2006-09-26 出生)

---

## 一、项目概述

### 1.1 目标

为不懂代码的用户（小白）构建一个 **PWA（渐进式网页应用）**，在 **安卓手机** 上为主、**Windows 电脑** 上也能查看，每天早上自动推送当天的星座运势和黄历信息，用户无需自己上网搜寻整合。

### 1.2 核心理念

- **简洁高级**：白色基底 + 清新绿色点缀，清爽有质感
- **真实数据**：从互联网免费渠道获取运势和黄历数据，不涉及 AI 生成
- **轻量化**：星座信息一行带过，运势用一两句话简短概括，拒绝复杂星级评分
- **被动接收**：每天早上固定时间推送，用户打开即看

---

## 二、技术方案

| 项目 | 选择 |
|------|------|
| 架构 | PWA (Progressive Web App) |
| 手机端 | 浏览器"添加到主屏幕"，像原生 App 一样使用 |
| 电脑端 | 打开浏览器直接访问 |
| 推送 | 浏览器推送通知（Service Worker） |
| 数据 | 免费网络 API / 网页抓取 |
| 部署 | 免费静态托管（如 Vercel / Netlify / GitHub Pages） |

## 三、页面设计

### 3.1 今日首页

**页面结构（从上到下）：**

1. **状态栏** — 时间、电量等
2. **日期区** — 公历日期 + 星期 + 农历日期
3. **星座** — ⚖️ 天秤座 · LIBRA（一行轻量化显示）
4. **今日总评** — 绿色渐变卡片，一段话概括当天整体运势
5. **爱情运势** — 粉色图标 + 一句话介绍
6. **财运运势** — 金色图标 + 一句话介绍
7. **黄历** — 三列：宜 ✅ / 忌 ❌ / 冲煞 🐯
8. **幸运信息** — 幸运数字 · 幸运颜色 · 吉时（横向一排）
9. **底部导航** — 📖 今日 / 📅 过去一周 / ⚙️ 设置

**设计风格：**
- 背景色：柔和的浅绿渐变（`#f6faf6 → #edf3ed`）
- 卡片：白色圆角 + 浅绿边框 + 柔和阴影
- 强调色：墨绿色（`#3d6b3d` / `#5b8c5b`）
- 顶部总评卡片使用绿色渐变

### 3.2 过去一周页面

**页面结构：**
1. **本周整体概括** — 绿色渐变卡片，一句话总结过去一周
2. **每日运势列表** — 7 张卡片，按日期排列
   - 左侧：星期 + 月/日
   - 中间：一句话运势描述
   - 右侧：天气表情符号表示运势趋势（☀️ 好 / ⛅ 普通 / 🌥️ 一般）
3. **今天高亮标记** — 当天卡片用浅绿背景突出显示
4. **底部导航**

### 3.3 设置页面

**分组设置项：**

| 分组 | 设置项 | 类型 | 说明 |
|------|--------|------|------|
| 推送设置 | 每日推送 | 开关 | 开启/关闭每日推送 |
| 推送设置 | 推送时间 | 选择 | 设置推送时间（默认 7:30） |
| 内容设置 | 我的星座 | 选择 | 切换查看运势的星座 |
| 内容设置 | 显示农历 | 开关 | 首页是否显示农历日期 |
| 其他 | 数据来源 | 跳转 | 查看数据来源信息 |
| 其他 | 关于 | 跳转 | 版本号等信息 |

## 四、推送通知

**样式：** 简洁版

**内容格式：**
> 爱情运不错，有机会遇到心动的人 💕 · 财运注意控制开销

用户收到推送后，点击可打开 App 查看完整内容。

**推送时机：** 每天早上用户设定的时间（默认 7:30）

---

## 五、数据需求

### 5.1 星座运势数据

从免费渠道获取以下信息：
- 当日整体运势描述（一段话）
- 爱情运势（一句话 + 评分/等级）
- 财运运势（一句话 + 评分/等级）
- 幸运数字
- 幸运颜色
- 历史运势数据（用于过去一周回顾）

### 5.2 黄历数据

从免费渠道获取：
- 公历 → 农历日期转换
- 当日宜/忌事项
- 冲煞生肖
- 吉时

---

## 六、非功能性需求

- **离线可用**：PWA 支持 Service Worker 缓存，无网络时也能查看最近一次运势
- **首次加载**：按需加载，首页应快速呈现
- **推送权限**：首次使用时请求通知权限
- **响应式**：适配手机和电脑屏幕
